	Выполнение SQL-запроса функцией "setquery()". Функция получает аргументом строку запроса, возвращает при неудаче false при удаче true... Если выполнен запрос select, то возвращается его порядковый номер в массиве...
	Вывод размера массива функцией "outcount()". Функция получает аргумент $querynum, определяющий номер массива. Если $querynum не задан, то выводится размер массива последнего запроса select.
	Вывод результата запроса select с помощью ф-ции "out()". Функция получает 3 аргумента $num, $nam, $querynum. $querynum - номер запроса в массиве; $num - номер строки в запросе; $nam - имя переменной из бд. Аргумент $querynum - не обязателен, если он не задан - используется результат последнего запроса. Функция возвращает значение конкретной переменной.
